1️⃣ Objective

Build a consumer-facing web application that answers:

“If my income stops or drops, how long will my savings last?”

This is NOT a statutory redundancy calculator.
This is a financial runway / sustainability modelling tool.

It must:

	•	Be premium-feeling

5️⃣ Engine Logic

Step 1: Monthly Burn Rate

For each month:

If month <= monthsUntilNewJob:
income = replacementMonthlyIncome + benefitSupportEstimate
Else:
income = assumed new income

NetBurn = totalExpenses - income

If NetBurn < 0:
surplus = abs(NetBurn)
Else:
deficit = NetBurn

⸻

Step 2: Capital Projection

Simulate month-by-month:

capital(t+1) = capital(t) - deficit + surplus

Stop when:
capital <= 0

Output:
	•	monthsUntilDepletion
	•	capitalAfter3Months
	•	capitalAfter6Months
	•	capitalAfter12Months
	•	depletionDate (optional)

Cap projection at 60 months for MVP.

⸻

Step 3: Scenario Comparison (MVP = 3 scenarios)

Scenario A: Income = 0
Scenario B: 50% income
Scenario C: New job after X months

Allow user to adjust X.

Display side-by-side cards:
	•	Runway months
	•	Remaining capital at 12 months
	•	Stability band

⸻

6️⃣ Stability Scoring (Deterministic)

Start score = 100

Deductions:
	•	< 3 months runway → -40
	•	< 6 months runway → -25
	•	Mortgage > 40% of essential expenses → -10
	•	Debt repayments > 25% of total expenses → -10
	•	Replacement income < 50% of original → -10

Band:

80–100 = Stable
60–79 = Watch
<60 = High Pressure

No subjective logic.
